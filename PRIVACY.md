# Политика конфиденциальности — Cookie Cleaner

**Дата вступления в силу:** 16 июня 2026 г.

## Сбор данных

Расширение **Cookie Cleaner** **не собирает, не хранит и не передаёт** какие-либо личные данные пользователя.

## Используемые разрешения

- `cookies` — необходимо для чтения и удаления куки-файлов браузера
- `activeTab` — необходимо для определения URL текущей вкладки
- `<all_urls>` — необходимо для удаления кук на любых сайтах

Все операции выполняются **локально** в браузере пользователя. Расширение не имеет серверной части, не отправляет данные по сети и не использует внешние API.
